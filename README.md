## Installation

Just add it to your ``rebar.config`` deps:

```erlang
  {deps, [
    ...
    {laotzi-demo, ".*",
      {git, "git@github.com:YOURNAME/laotzi-demo.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
